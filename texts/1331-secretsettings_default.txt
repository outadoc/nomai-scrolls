// Uncomment this line to set the time period used for the mouse smoothing algorithm.
// By default, the time period is 1/6 sec. Set the time period to 0 in order to turn
// off mouse smoothing, increase the time to amplify the effect.
// 
// For the math, the weight of a given mouse delta input is given by the following 
// half life equation for all mouse inputs from time 0 to T:
// 2 ^ -(t / T * 10) where T = MouseSmoothTimeBuffer and t is the age of the mouse input
// 
// Prior to Patch 15, this time period was not set in seconds but in frames, and 
// this resulted in varying mouse smoothing effects depending on the user's framerate.
// A higher framerate would result in basically no smoothing at all, whereas lower
// framerates would have stronger mouse smoothing.
//
// Default value: 0.166667
// Valid values: Float values 0 and greater
//MouseSmoothTimeBuffer=0.166667
